# 📖 Guide d'Utilisation du CMS - Les Bains de l'Opéra

Ce guide explique comment modifier le contenu de votre site web facilement, sans connaissances techniques.

## 🚀 Démarrage

### Accéder à l'interface d'administration

1. Ouvrez votre navigateur (Chrome, Firefox, Safari, etc.)
2. Allez sur votre site : `https://votre-site.com/login.html`
3. Entrez vos identifiants :
   - **Nom d'utilisateur** : admin
   - **Mot de passe** : (votre mot de passe)
4. Cliquez sur **Se connecter**

Vous arrivez sur la page de gestion des réservations. Pour gérer le contenu du site :
- Cliquez sur le bouton **"Gérer le contenu du site"** en haut à droite

## 📝 Que pouvez-vous modifier ?

Le système vous permet de modifier 4 types de contenus :

### 1. 🔧 Paramètres Généraux

C'est ici que vous gérez toutes les informations de base du site.

**Ce que vous pouvez modifier :**
- Nom du site
- Description
- Adresse complète
- Email de contact
- Téléphone
- **Horaires d'ouverture** pour chaque jour de la semaine

**Comment faire :**
1. Cliquez sur l'onglet **"Paramètres"**
2. Modifiez les champs que vous souhaitez changer
3. Cliquez sur **"Enregistrer les paramètres"** en bas

**💡 Astuce :** Pour les horaires, vous pouvez écrire "Fermé" ou les horaires comme "11h - 19h"

---

### 2. 🖼️ Carousel (Images de la page d'accueil)

Le carousel affiche les grandes images qui défilent sur la page d'accueil.

**Comment ajouter une nouvelle image :**
1. Cliquez sur l'onglet **"Carousel"**
2. Cliquez sur **"Ajouter une image"**
3. Remplissez le formulaire :
   - **URL de l'image** : `images/nom-de-votre-image.jpg` (l'image doit d'abord être uploadée dans le dossier images)
   - **Titre** : Le grand titre qui apparaît sur l'image (ex: "MASSAGES & SOINS")
   - **Sous-titre** : Le texte plus petit (ex: "Découvrez nos rituels")
   - **Texte du bouton** : Ce qui est écrit sur le bouton (ex: "Réserver")
   - **Lien du bouton** : Vers quelle page le bouton mène (ex: "reservation.html")
   - **Ordre d'affichage** : 1, 2, 3, 4... (dans quel ordre l'image apparaît)
4. Cliquez sur **"Enregistrer"**

**Comment modifier une image existante :**
1. Dans le tableau, cliquez sur **"Modifier"** à côté de l'image
2. Changez les informations
3. Cliquez sur **"Enregistrer"**

**Comment supprimer une image :**
1. Cliquez sur **"Supprimer"** à côté de l'image
2. Confirmez la suppression

---

### 3. 💰 Formules & Tarifs

Gérez toutes vos prestations : hammam, massages, soins, escales, etc.

**Comment ajouter une nouvelle formule :**
1. Cliquez sur l'onglet **"Formules & Tarifs"**
2. Cliquez sur **"Ajouter une formule"**
3. Remplissez le formulaire :
   - **Nom** : Le nom de la formule (ex: "MASSAGE RELAXANT 60MIN")
   - **Type** : Choisissez parmi :
     - Escale
     - Escale avec SPA
     - Hammam
     - Massage
     - Soin
   - **Description** : Décrivez la prestation (ex: "Massage relaxant des pieds à la tête")
   - **Prix** : Le prix en euros (ex: 65)
   - **Durée** : La durée en minutes (ex: 60)
4. Cliquez sur **"Enregistrer"**

**Comment modifier une formule :**
1. Dans le tableau, cliquez sur **"Modifier"** à côté de la formule
2. Changez les informations (prix, description, durée, etc.)
3. Cliquez sur **"Enregistrer"**

**Comment supprimer une formule :**
1. Cliquez sur **"Supprimer"** à côté de la formule
2. Confirmez la suppression

⚠️ **Attention :** Si vous supprimez une formule, les clients ne pourront plus la réserver.

---

### 4. 🎁 Offres Spéciales

Créez des promotions temporaires qui s'affichent sur la page d'accueil.

**Comment créer une nouvelle offre :**
1. Cliquez sur l'onglet **"Offres Spéciales"**
2. Cliquez sur **"Ajouter une offre"**
3. Remplissez le formulaire :
   - **Titre** : Le nom de l'offre (ex: "OFFRE FÊTE DES MÈRES")
   - **Description** : Décrivez l'offre (ex: "Hammam + Massage 30min pour 2 personnes")
   - **Prix** : Le prix promotionnel (ex: 80)
   - **Prix barré** : L'ancien prix (optionnel, ex: 100) - il sera affiché barré
   - **Valide du** : Date de début de l'offre (optionnel)
   - **Valide jusqu'au** : Date de fin de l'offre (optionnel)
4. Cliquez sur **"Enregistrer"**

**Comment modifier une offre :**
1. Cliquez sur **"Modifier"** à côté de l'offre
2. Changez les informations
3. Cliquez sur **"Enregistrer"**

**Comment supprimer une offre :**
1. Cliquez sur **"Supprimer"** à côté de l'offre
2. Confirmez la suppression

💡 **Astuce :** Si vous ne mettez pas de date de fin, l'offre sera affichée en permanence jusqu'à ce que vous la supprimiez.

## ❓ Questions Fréquentes

### Comment mettre à jour les horaires pour les vacances ?

1. Allez dans **"Paramètres"**
2. Modifiez les horaires des jours concernés
3. Mettez "Fermé" pour les jours de fermeture
4. Cliquez sur **"Enregistrer les paramètres"**

### Comment créer une offre pour le mois en cours ?

1. Allez dans **"Offres Spéciales"**
2. Cliquez sur **"Ajouter une offre"**
3. Remplissez le titre (ex: "Offre de Mars")
4. Mettez la description de l'offre
5. Dans **"Valide jusqu'au"**, mettez le dernier jour du mois
6. Cliquez sur **"Enregistrer"**

L'offre s'affichera automatiquement sur la page d'accueil et disparaîtra après la date de fin.

### Comment changer le prix d'une formule ?

1. Allez dans **"Formules & Tarifs"**
2. Trouvez la formule à modifier dans le tableau
3. Cliquez sur **"Modifier"**
4. Changez le prix
5. Cliquez sur **"Enregistrer"**

Le nouveau prix sera affiché immédiatement sur le site.

### Comment ajouter une nouvelle image sur la page d'accueil ?

1. D'abord, uploadez votre image dans le dossier `images` de votre site (demandez à votre administrateur technique si besoin)
2. Allez dans **"Carousel"**
3. Cliquez sur **"Ajouter une image"**
4. Dans "URL de l'image", écrivez `images/nom-de-votre-image.jpg`
5. Remplissez le titre, sous-titre, etc.
6. Cliquez sur **"Enregistrer"**

## 🆘 En cas de problème

### Je ne peux pas me connecter
- Vérifiez que vous utilisez le bon nom d'utilisateur et mot de passe
- Essayez de fermer et rouvrir votre navigateur
- Contactez votre administrateur technique

### Mes modifications n'apparaissent pas sur le site
- Rafraîchissez la page du site (touche F5 ou Ctrl+R / Cmd+R)
- Videz le cache de votre navigateur
- Attendez quelques minutes que les modifications se propagent

### J'ai fait une erreur, comment annuler ?
- Retournez dans la section concernée
- Cliquez sur "Modifier" pour corriger
- Ou cliquez sur "Supprimer" si vous voulez tout enlever

### Le bouton "Enregistrer" ne fonctionne pas
- Vérifiez que tous les champs obligatoires sont remplis
- Vérifiez que les prix et durées sont bien des nombres
- Essayez de rafraîchir la page et recommencer
